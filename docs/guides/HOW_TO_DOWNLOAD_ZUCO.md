## â“ Troubleshooting

### Problem: OSF Download is Slow
**Solution:** 
- Download during off-peak hours
- Use download manager with resume capability
- Download one task at a time

### Problem: Files are Corrupted
**Solution:**
- Re-download the file
- Check file size matches expected
- Run verification script

### Problem: Can't Access OSF
**Solution:**
- OSF may require account registration (free)
- Create account at https://osf.io/register/
- Some datasets require access request

### Problem: Different File Structure
**Solution:**
The verification script is flexible. As long as you have `.mat` files in `data/raw/zuco/`, it should work. The exact folder names don't matter.

## ğŸ“Š What You're Downloading

**ZuCo Dataset Details:**
- **Subjects:** 12 native English speakers
- **Tasks:** 3 reading paradigms
- **Channels:** 105 EEG channels
- **Sampling Rate:** 500 Hz
- **Sentences:** ~9,000 total
- **Size:** ~12-15 GB total
- **Format:** MATLAB .mat files

**Citation:**
```
Hollenstein, N., Rotsztejn, J., Troendle, M., Pedroni, A., Zhang, C., & Langer, N. (2018).
ZuCo, a simultaneous EEG and eye-tracking resource for natural sentence reading.
Scientific data, 5(1), 1-13.
```
